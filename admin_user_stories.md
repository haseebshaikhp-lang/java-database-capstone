# User Story 1

## Story Title
Admin manages doctors

### As an
Admin

### I want
to add, update, delete, and view doctor records

### So that
I can efficiently manage doctors in the Smart Clinic system

## Acceptance Criteria
- [ ] Admin can add a new doctor.
- [ ] Admin can update doctor details.
- [ ] Admin can delete a doctor.
- [ ] Admin can view the list of all doctors.

## Notes
- Only authenticated admins can perform these actions.
- Doctor information must be validated before saving.
